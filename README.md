# PAV-MC

**PAV-MC** is a Fabric 1.21.11 modpack built for long-term multiplayer survival. It expands vanilla Minecraft with new biomes, structures, combat mechanics, enchantments, equipment, exploration, and quality-of-life improvements while maintaining the familiar Minecraft experience.

Designed for our community server, PAV-MC focuses on rewarding exploration, meaningful progression, and cooperative gameplay without overwhelming players with unnecessary complexity.

---

## 🌍 Explore a Bigger World

The world is larger, more diverse, and far more rewarding to explore.

### World Generation

* **Biomes O' Plenty** adds dozens of unique biomes.
* **Terralith** completely redesigns terrain generation with impressive mountains, valleys, caves, and landscapes.
* **ChoiceTheorem's Overhauled Village** and **Towns & Towers** greatly improve villages and naturally generated structures.
* **Enderscape** transforms The End into a fully realized dimension with new biomes, creatures, resources, and locations to discover.

Exploration remains exciting throughout the entire game instead of becoming repetitive after finding a few structures.

---

## ⚔️ Enhanced Combat

Combat has been modernized and expanded with new mechanics and equipment.

Features include:

* Improved attack animations with **Better Combat**
* Multiple new ranged weapons with **Too Many Bows**
* Firearms through **Guns++**
* Dozens of new enchantments from **Beyond Enchant** and **Enchantments Expansion**
* Controlled enchanting using **Enchanting Infuser**

Players have significantly more freedom when creating combat builds instead of relying on vanilla gear alone.

---

## 🛡️ Expanded Progression

Progression continues well beyond vanilla Minecraft.

* Upgrade equipment further with **Advanced Netherite**.
* Discover powerful enchantments.
* Find structures using **Explorer's Compass**.
* Locate rare biomes using **Nature's Compass**.

These additions provide meaningful goals long after defeating the Ender Dragon.

## 🎙️ Multiplayer Focused

PAV-MC was designed around cooperative multiplayer.

Features include:

* Individual loot chests using Lootr
* Built-in proximity voice chat with Simple Voice Chat
* Fair exploration rewards for every player
* Shared progression without competing for generated loot

Whether you're building a town, exploring together, or tackling difficult structures, the pack is built to support long-term community play.

---

## 🚀 Performance Optimized

Despite adding substantial new content, PAV-MC includes several optimization mods to improve frame rates and server performance.

Performance enhancements include:

* Sodium
* Lithium
* FerriteCore
* Noisium
* Distant Horizons support
* Clumps

These mods reduce memory usage, improve chunk loading, and increase overall performance while maintaining visual quality.

The pack also enhances Minecraft's atmosphere with optional visual improvements.
